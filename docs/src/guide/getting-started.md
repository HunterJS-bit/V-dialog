# Getting Started


## Prerequisites

- [Vue](https://vuejs.org/)



## Manual Installation


- **Step 1**: Install package

#####  NPM
```shell
npm install -S v-dialog
```

##### Yarn
```shell
yarn add v-dialog
```


- **Step 2**: Import it and use it in your project

```
import Vdialog from 'v-dialog';


```


Please check our [Examples section](./examples.md) for more usages of the dialog
